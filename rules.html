3.1.1 Учётная запись Продавца

После создания учётной записи «Продавец» Пользователю предлагается приобрести один из пакетов услуг. Для тарифа «Лёгкий» ограничение по времени не применяется, все остальные тарифы действительны в течение 30 дней с момента активации.

— Пакет «Лёгкий» — покупка определённого количества Сделок для одного товара.

— Пакет «Стандарт» — включает в себя размещение до 5 товаров с общим количеством 250 Сделок.

— Пакет «Продвинутый» — включает в себя размещение до 12 товаров с общим количеством 600 Сделок.

— Пакет «Король маркетплейсов» — включает в себя размещение до 25 товаров с общим количеством 1250 Сделок.

После оплаты тарифа откроется доступ в ЛК, который включает в себя весь необходимый интерфейс для создания и проведения Сделок.

3.1.2 Типы рекламных кампаний

На Сервисе Продавец может разместить предложения о продаже товаров, используя по своему усмотрению несколько типов рекламных кампаний — «Кешбэк без отзыва», «Кешбэк после Отзыва» и «Купон».

  • «Кешбэк без отзыва» — рекламная кампания, направленная на увеличение продаж и трафика на карточку товара. Предусматривает получение Кешбэка Покупателем.
  • «Кешбэк с отзывом» — предусматривает получение Кешбэка Покупателем после написания Покупателем отзыва. Срок возврата Кешбэка (выставляется Продавцом при создании товара) не может превышать 21 день с момента получения товара Покупателем. Устанавливая срок возврата Кешбэка менее срока возврата товара на торговой площадке, Продавец отказывается от претензий в случае осуществления возврата его товара после получения Кешбэка Покупателем.
  • «Купон» — предусматривает получение скидки на товар путём применения промокода на торговой площадке, с последующим написанием отзыва Покупателем. Данная рекламная кампания предусматривает получение Баллов внутри Сервиса, позволяющих совершать больше покупок. Подходит только для площадок, где предусмотрено создание купонов и промокодов. Для данного типа сделки не требуется пополнения Депозита.
3.1.3 Создание рекламной кампании

Продавец самостоятельно через ЛК создаёт рекламную кампанию на Сервисе. В рекламной кампании Продавец описывает условия участия в данной рекламной кампании и предоставляет информацию о товаре, такую как: наименование, описание, цена, размер Кешбэка, изображения, ссылка на товар (по умолчанию используется только для модерации; по желанию Продавца доступна Покупателю для покупки по прямой ссылке), торговую площадку, срок возврата Кешбэка. Рекламная кампания после модерации появится на сайте в виде карточки товара, которая автоматически дублируется в Телеграмм - группе для Покупателей. Продавец, публикуя на Сервисе рекламную кампанию с определёнными им условиями, соглашается, что количество желающих заключить Сделку находится в прямой зависимости от размера Кешбэка, а именно: чем выше размер Кешбэка, тем больше желающих приобрести товар.
После получения заявки на Сделку Продавец самостоятельно управляет её ходом и контролирует выполнение условий Сделки. Для участия в Сделке Продавцу необходимо пополнить Депозит (кроме сделки с Купоном). Депозит является гарантией для Сервиса и Покупателя, что последний получит свой Кешбэк (подробнее в разделе 3.1.4.). При одобрении запроса на Сделку, на Депозите замораживаются денежные средства в размере указанного в карточке товара размера Кешбэка, и размораживаются при завершении Сделки.

3.1.4 Депозит

Депозит необходим для предотвращения случаев неисполнения взятых Продавцом на себя обязательств перед Покупателем. Сумма пополнения Депозита и вывода с Депозита должна быть не менее 1000 рублей. Все денежные средства на Депозите доступны Продавцу для вывода, за исключением «замороженных» средств в Активных сделках.

Денежные средства «замораживаются» по каждой Активной сделке и «размораживаются» по завершению таковой. Денежные средства на Депозите требуются для рекламных кампаний типа «Кешбэк После Отзыва» и «Кешбэк После Доставки» (без отзыва). Для возврата частично или полностью денежных средств, внесённых Продавцом на Депозит посредством платёжной системы Тинькофф, Продавцу необходимо воспользоваться функционалом Личного Кабинета (вкладка «Депозит»). Заявка на возврат обрабатывается ДК в течении семи рабочих дней. Возврат производится ДК после обработки заявки на вывод Депозита путём полного или частичного возврата средств по операции (операциям) в сроки, установленные правилами платёжной системы Тинькофф и банка-эмитента карты Продавца, при условии выполнения обязательств в полном объёме перед Покупателями.
